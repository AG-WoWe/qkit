
Circlefit Version 3.1  (04.06.2015)
by Sebastian Probst, email: sebastian.probst@kit.edu

Software requirements:
  * Python 2.7
  * Python libraries:
    * numpy
    * scipy
    * matplotlib (required by the test script)

--- New in version 3.1 compared to 3.0 --------
 * consequent use of numpy arrays and numpy functions
 * library numdifftools is no longer required
 * speedup of the error calculation by a factor ~100
 * speedup of circlefit subroutine by a factor of ~25
